## 🔧 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/zenvix-solutions.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start local development:**
   ```bash
   npm start
   # or
   yarn start
   ```
